# 安全说明

## 数据隐私

「破壁视界」是一款完全本地运行的桌面应用：

- **无云端存储**：所有灵感记录、配置、语音文件均存储在本地设备
- **无数据上传**：除非你主动触发 AI API 调用，否则不会向任何服务器发送数据
- **API Key 安全**：你的 API Key 仅存储在本地 localStorage 和配置文件中，不上传任何第三方

## AI API 调用

当你使用「破壁分析」或「灵感火花」功能时，应用会调用你配置的 AI API：

- 仅发送当前触发的文本内容
- 不发送历史记录
- 不发送个人身份信息

## 安全最佳实践

1. **不要分享 API Key**：你的 API Key 相当于密码，请勿在 Issue、PR 或任何公开场合泄露
2. **使用最小权限**：如果支持，为 API Key 设置最小权限范围
3. **定期轮换**：定期更换你的 API Key，特别是在怀疑泄露时
4. **注意网络安全**：仅在安全网络环境下使用应用

## 漏洞报告

如果你发现任何安全漏洞，请通过以下方式报告：

1. 在 GitHub Issues 中提交（标记 `security` label）
2. 描述漏洞的影响范围和复现步骤
3. 不要在公开场合讨论具体漏洞细节

我们会在收到报告后尽快处理。
