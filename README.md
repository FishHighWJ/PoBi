# 破壁视界

一款浮在屏幕上的轻量桌面工具（Electron 应用），以悬浮窗形式常驻桌面，无需安装，双击即可使用，随时记录灵感，并提供破壁功能打破信息茧房。

## 功能特性

- 🪟 **悬浮窗**：常驻桌面顶部的浮动工具栏，方便随时调用
- 📝 **灵感记录**：快速记录想法和灵感，支持分类管理
- 🎙️ **语音输入**：支持语音转文字，解放双手
- 🌓 **暗色模式**：支持亮色/暗色主题切换
- 📊 **每日/每周回顾**：查看记录统计和回顾
- ✨ **灵感火花**：随机回顾过去的灵感记录

## 文件说明

| 文件 | 说明 |
|------|------|
| `破壁视界-发布版.zip` | 正式发布版（Electron 打包），解压后双击运行 |
| `破壁视界-Demo.zip` | 轻量 Demo 版 |
| `demo.html` | 网页版 Demo，直接在浏览器中打开即可体验 |

## 快速开始

### 发布版
1. 下载 `破壁视界-发布版.zip`
2. 解压到任意目录
3. 双击 `破壁视界.exe` 运行

### 网页 Demo
直接在浏览器中打开 `demo.html` 即可体验核心功能。

## 技术栈

- Electron
- HTML / CSS / JavaScript
- Whisper API（语音识别）
